# Invasive Fish Reporter Malaysia

## Project Overview
This project develops a GIS-integrated mobile application for monitoring invasive freshwater fish species in Malaysia.

## Objectives
- To allow users to report invasive fish sightings
- To integrate GIS for real-time mapping
- To validate reported species using image-based identification

## Technologies Used
- Mobile Development
- GIS Integration
- Image Recognition
- Database Management System
